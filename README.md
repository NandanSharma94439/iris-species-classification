# 🌸 Iris Species Classification

A machine learning classification project that predicts the species of Iris flowers using sepal and petal measurements. The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

---

## 📌 Overview

The Iris dataset is one of the most widely used datasets for supervised learning. This project applies machine learning algorithms to classify Iris flowers into three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica
